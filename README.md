# 🤖 Caja de Herramientas de Robot en Go

Este proyecto es una aplicación de línea de comandos (CLI) en **Go** que simula el comportamiento de un robot al realizar una serie de tareas. Utiliza un menú interactivo para que los usuarios puedan probar las "habilidades" del robot, como el cálculo matemático rápido y el análisis de texto sin emociones.

## 🚀 Características

-   **Menú interactivo**: Navega por diferentes funciones con una interfaz simple.
-   **Prueba de velocidad**: Resuelve una operación matemática compleja y mide el tiempo de procesamiento.
-   **Análisis objetivo**: Lee y resume una historia, eliminando cualquier contexto emocional.
-   **Lógica literal**: Resuelve un acertijo basándose puramente en la lógica, sin humor.
-   **Búsqueda eficiente**: Identifica una palabra en un texto largo de manera rápida.
-   **Respuesta consistente**: Responde a múltiples preguntas en un tiempo de respuesta fijo y exacto.

## 🛠️ Requisitos

-   **Go** y **Goc** instalados. Puedes obtenerlos desde la [página oficial de Go](https://go.dev/doc/install).

## 🏃 Modo de uso

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/santiagourdaneta/Caja-de-Herramientas-de-Robot-en-Go/
    cd Caja-de-Herramientas-de-Robot-en-Go
    ```
2.  Ejecuta el programa:
    ```bash
    go run main.go
    ```
3.  El menú principal aparecerá. Simplemente elige una opción y presiona Enter para interactuar con la caja de herramientas del robot. Para salir, elige la opción `0`.
